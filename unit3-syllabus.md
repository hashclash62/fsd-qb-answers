Unit 3: Security, Authentication, and Backend Testing Security & Authentication:
Introduction to Spring Security and implementation of JWT (JSON Web Tokens) for
authentication and authorization. Encrypting passwords using BCrypt Password Encoding
and implementing role-based access control (RBAC).
Backend Testing:
Understanding unit testing in Java and writing JUnit test cases for Service and Repository
layers. Using Mockito for mocking dependencies and testing RESTful APIs using Spring
Boot‘s MockMvc. Implementing global exception handling using @ControllerAdvice.
Logging with SLF4J and Logback, optimizing API performance with caching, and profiling
and monitoring Spring Boot applications.
